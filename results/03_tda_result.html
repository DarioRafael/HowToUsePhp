TIPOS DE DATOS ABSTRACTOS EN PHP

Los Tipos de Datos Abstractos (TDA) son estructuras que encapsulan datos y operaciones relacionadas. En PHP, se implementan mediante clases y objetos, siguiendo los principios de la Programación Orientada a Objetos.

1. TDA PUNTO

class Punto { // Propiedades (atributos) del TDA private $x; private $y; // Constructor: inicializa el estado del objeto public function __construct($x = 0, $y = 0) { $this->x = $x; $this->y = $y; } // Getters y Setters public function getX() { return $this->x; } public function getY() { return $this->y; } public function setX($x) { $this->x = $x; } public function setY($y) { $this->y = $y; } // Método para calcular la distancia a otro punto public function distanciaA(Punto $otroPunto) { $dx = $this->x - $otroPunto->getX(); $dy = $this->y - $otroPunto->getY(); return sqrt(pow($dx, 2) + pow($dy, 2)); } // Método para representar el punto como string public function toString() { return "($this->x, $this->y)"; } }

Características del TDA Punto:

  • Atributos: Coordenadas x e y (private)
  • Constructor: Inicializa las coordenadas
  • Métodos de acceso: Getters y setters para cada coordenada
  • Métodos de operación: Cálculo de distancia entre puntos

⭐ Ejemplo de uso del TDA Punto

// Crear dos instancias de Punto
$punto1 = new Punto(3, 4);
$punto2 = new Punto(7, 9);

// Mostrar los puntos
echo "Punto 1: " . $punto1->toString();
echo "Punto 2: " . $punto2->toString();

// Calcular la distancia entre los puntos
echo "Distancia: " . $punto1->distanciaA($punto2);

// Modificar las coordenadas del punto 1
$punto1->setX(5);
$punto1->setY(6);
echo "Punto 1 modificado: " . $punto1->toString();
Resultado:
Punto 1: (3, 4)
Punto 2: (7, 9)
Distancia entre puntos: 6.4031242374328

Punto 1 modificado: (5, 6)
Nueva distancia entre puntos: 3.605551275464

2. TDA PILA (STACK)

class Pila { // Atributos private $elementos = []; private $tamañoMax; // Constructor public function __construct($tamañoMax = 10) { $this->tamañoMax = $tamañoMax; } // Método para añadir un elemento a la pila (push) public function push($elemento) { if (count($this->elementos) < $this->tamañoMax) { array_push($this->elementos, $elemento); return true; } return false; // Pila llena } // Método para sacar un elemento de la pila (pop) public function pop() { if (!$this->estaVacia()) { return array_pop($this->elementos); } return null; // Pila vacía } // Método para ver el elemento en la cima sin sacarlo (peek) public function peek() { if (!$this->estaVacia()) { return end($this->elementos); } return null; } // Método para comprobar si la pila está vacía public function estaVacia() { return empty($this->elementos); } // Método para obtener el tamaño actual de la pila public function tamaño() { return count($this->elementos); } // Método para representar la pila como string public function toString() { return implode(", ", array_reverse($this->elementos)); } }

Características del TDA Pila:

  • Principio LIFO: Last In, First Out (Último en entrar, primero en salir)
  • Atributos: Arreglo de elementos y tamaño máximo
  • Operaciones principales:
    • push(): Añade un elemento a la cima
    • pop(): Elimina y devuelve el elemento de la cima
    • peek(): Consulta el elemento de la cima sin eliminarlo

Estructura de una pila

⭐ Ejemplo de uso del TDA Pila

// Crear una pila con tamaño máximo de 5
$pila = new Pila(5);

// Añadir elementos
$pila->push(10);
$pila->push(20);
$pila->push(30);

// Mostrar la pila y su estado
echo "Elementos en la pila: " . $pila->toString();
echo "Tamaño actual: " . $pila->tamaño();
echo "Elemento en la cima: " . $pila->peek();

// Sacar un elemento
$elementoSacado = $pila->pop();
echo "Elemento sacado: $elementoSacado";
echo "Elementos restantes: " . $pila->toString();

3. TDA COLA (QUEUE)

class Cola { private $elementos = []; // Método para añadir un elemento al final de la cola (enqueue) public function enqueue($elemento) { array_push($this->elementos, $elemento); } // Método para sacar un elemento del frente de la cola (dequeue) public function dequeue() { if (!$this->estaVacia()) { return array_shift($this->elementos); } return null; } // Método para ver el elemento al frente sin sacarlo public function front() { if (!$this->estaVacia()) { return $this->elementos[0]; } return null; } // Método para comprobar si la cola está vacía public function estaVacia() { return empty($this->elementos); } // Método para obtener el tamaño de la cola public function tamaño() { return count($this->elementos); } // Método para representar la cola como string public function toString() { return implode(", ", $this->elementos); } }

Características del TDA Cola:

  • Principio FIFO: First In, First Out (Primero en entrar, primero en salir)
  • Atributos: Arreglo de elementos
  • Operaciones principales:
    • enqueue(): Añade un elemento al final de la cola
    • dequeue(): Elimina y devuelve el elemento del frente
    • front(): Consulta el elemento del frente sin eliminarlo

Estructura de una cola

⭐ Ejemplo de uso del TDA Cola

// Crear una cola
$cola = new Cola();

// Añadir elementos a la cola
$cola->enqueue("A");
$cola->enqueue("B");
$cola->enqueue("C");

// Mostrar la cola y su estado
echo "Cola: " . $cola->toString();
echo "Elemento al frente: " . $cola->front();

// Sacar un elemento
$elementoSacado = $cola->dequeue();
echo "Elemento sacado: $elementoSacado";
echo "Cola actualizada: " . $cola->toString();
